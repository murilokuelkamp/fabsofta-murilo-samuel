    ### Revisão de Java

    - JRE - Java Runtime Environment
        - Máquina vitual do Java - interpreta o Bytecode(.class) do Java para a linguagem de máquina
    - JDK - Java Development Kit (javac.exe)
        - Compilar o código Java em bytecode(.class)
